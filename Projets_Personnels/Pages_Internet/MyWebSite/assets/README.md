# Assets requis

Placez dans ce dossier :

- **photoprofil.jpg** — votre photo de profil (recommandé : 400×400px minimum, format carré)
- **CV_Aurelien_Logeais.pdf** — votre CV au format PDF (référencé par le bouton "Télécharger CV")

Le site fonctionne sans ces fichiers (la photo s'efface proprement si absente),
mais le bouton de téléchargement du CV mènera vers un 404.
