# 🖼️ Dossier des images de posts LinkedIn

Mets ici toutes les photos que tu veux joindre à tes posts.

## Convention de nommage recommandée

```
post-{id}-{description}.jpg
```

Exemples :
- `post-5-splunk-dashboard.jpg`
- `post-5-homelab-overview.png`
- `post-6-securitplus-badge.jpg`

## Formats acceptés
JPG, PNG, WebP — tous fonctionnent. Préfère JPG/WebP pour le poids.

## Taille recommandée
- Largeur max : 1200px (inutile de mettre plus grand)
- Compression : utilise https://squoosh.app/ si les fichiers sont lourds

## Référencement dans le JSON

```json
"images": ["data/posts-images/post-5-splunk-dashboard.jpg"]
```
